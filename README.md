## 🚀 Completed Projects

### 🌐 Task 1: Language Translation Tool

A modern AI-powered language translation application built with Python, Flask, and the Google Translate engine via deep-translator.

#### Key Features

* Translation across 20 languages
* Automatic language detection
* Speech-to-Text input
* Text-to-Speech output
* Translation history management
* Favorites system
* CSV and TXT export functionality
* Dark/Light mode support
* Responsive glassmorphism user interface

📂 Project Folder:


CodeAlpha_LanguageTranslator/

### 💬 Task 2: AI FAQ Chatbot

An intelligent FAQ chatbot powered by NLP techniques that understands user queries and provides the most relevant responses from a knowledge base containing 185 technical FAQs.

#### Key Features

* NLP-based question matching
* TF-IDF Vectorization
* Cosine Similarity Search
* Confidence Scoring
* Smart Suggestions
* Voice Input & Speech Output
* Favorites and Chat History
* Admin FAQ Management
* Dark/Light Theme
* Responsive Glassmorphism Interface

📂 Project Folder:

CodeAlpha_FAQChatbot/
